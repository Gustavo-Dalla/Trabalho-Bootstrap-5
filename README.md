# Portal de Notícias - Bootstrap 5

Este projeto é a página inicial de um portal de notícias responsivo, desenvolvido como um trabalho prático em HTML5 usando o framework Bootstrap 5.

---

## Recursos Implementados

O layout foi construído totalmente responsivo. Componentes do Bootstrap 5 utilizados de acordo com os requisitos do sistema:

*   **Navbar:** Barra de navegação fixa no topo, com logótipo, links, campo de pesquisa e comportamento colapsável (menu hambúrguer).
*   **Hero Section / Carousel:** Área de destaque principal, título e botão ("Saiba mais").
*   **Grid System:** Estrutura principal dividida em duas colunas no desktop (conteúdo principal `.col-lg-8` e barra lateral `.col-lg-4`).
*   **Cards de Conteúdo:** Grade com 6 cards de notícias com imagens, títulos, descrição e botões de ("Saiba mais").
*   **Modal Interativa:** Botão de login que aciona uma janela modal centralizada na tela.
*   **Formulário:** Formulário assinatura de newsletter.
*   **Footer:** Rodapé escuro com links úteis, direitos de autor e ícones de redes sociais.
